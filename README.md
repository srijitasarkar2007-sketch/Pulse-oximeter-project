# Pulse-oximeter-project
This is my first project in github. Its a project which helped me understand basics of circuit and the technology used in medical field .
This project is a pulse oximeter system that measures heart rate and SpO2 levels using the MAX30102 sensor. 

It uses Arduino and ESP32 for processing and displays the output on an OLED screen. This project helped me understand basic circuit design and biomedical sensing technology.
## Components Used
- Arduino Uno
- ESP32
- MAX30102 Sensor
- SSD1306 OLED Display
- Breadboard
- Jumper Wires and single core wires

## Working Principle
The MAX30102 sensor uses light absorption to measure oxygen levels in blood (SpO2) and heart rate. 

The data is processed using Arduino and displayed on an OLED screen and then the data is sent to Blynk dashboard via ESP32 , displaying the data in cloud.
## Features
- Real-time heart rate monitoring
- SpO2 measurement
- OLED display output
- Compact circuit design
